Firmware Upgrade Keypair Generation
-----------------------------------

This document lists the steps that were employed in the generation of the primary and secondary firmware upgrade keys used on the hub

### Generation of the keys

The keys used in this were generated by using a script as highlighted in the instructions below.

1. Clone the file-encryption repo.

```
> git clone ssh://git@git.intra.lutron.com:7999/lpfu/file-encryption.git
```

2. Run the script giving the public key name with the absolute path as an argument
/root/path/to/my/
```
> ./usr/lutron/create-asymmetric-keypair.sh /root/path/to/my/keyname
```

3. This will generate two files, a private key at `/root/path/to/my/keyname.pem` and a public key at `/root/path/to/my/keyname.pub`

4. This script was run with two arguments one for the primary and one for the secondary keys. The private keys of both have been stored in a secured encrypted disk and the public key is stored in this repo.
